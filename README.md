# AI Economy Explorer

Interactive knowledge graph of the AI economy — capital flows, compute commitments, semiconductor supply chains, power infrastructure. Live at <https://ai-economy.aktagon.com>.

This repository holds the deployed visualization (a single self-contained HTML page plus the projected graph data). The ontology, validation pipeline, and editorial YAML data live in a separate private repository.

## What you can do

Click an edge to see the relationship's parties, role labels, committed amount, evidence, and confidence tier. Use the view filters to switch between capital, compute, chips, power, optics, lithography, tools, and strategic projections of the same graph. Search to jump to a specific actor.

## Confidence and provenance

Every relationship cites at least one evidence source, ranked by reliability:

- `RegulatedTier` — SEC, ESEF, JPX, and equivalent regulator filings
- `CompanyTier` — press releases, official company announcements
- `JournalisticTier` — credible news reporting (FT, WSJ, Bloomberg, Reuters)
- `AnalysisTier` — analyst notes, run-rate estimates

The graph aligns with PROV-O — each evidence node is a `prov:Entity` with a `prov:wasAttributedTo` edge to the issuing actor.

## Disclaimer

Research project. Numbers are best-effort cites against public sources. Not investment advice, not authoritative, not real-time. Many edges represent announced commitments that may not fully execute.

## Build artifact

Files in this repo (`index.html`, `dist/graph.graphology.json`) are generated by the source pipeline and synced here on each deploy. Do not edit them in place — changes will be overwritten on the next deploy.
